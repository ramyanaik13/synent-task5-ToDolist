// Smart Task Manager
This application provides an easy way to manage daily tasks with priority and progress tracking.
This project is a **Smart Task Manager Web Application** developed as part of the **Synent Technologies Web Development Internship Program**.

// Project Description

Smart Task Manager is a modern task management application that helps users organize and track their daily activities efficiently. Users can create tasks, set priorities, add due dates, search tasks, filter tasks, and monitor their progress.

//Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome Icons
* Google Fonts

// Features

* Add new tasks
* Set task priority (High, Medium, Low)
* Add due dates for tasks
* Search tasks dynamically
* Mark tasks as completed
* Delete and manage tasks
* Filter tasks (All, Pending, Completed)
* Task completion progress tracking
* Dark mode support
* Responsive user interface
* Data persistence using localStorage

1. Clone or download the repository.
2. Open the project folder.
3. Open `index.html` in a web browser.
4. Start managing your tasks.
